    metrics

    metrics: WorldMetrics

    Numerical world state. The WorldMetrics fields below (population, morale, foodMonthsReserve, powerKw, etc.) are Mars/space heritage conveniences. Any scenario extends the bag via the [key: string]: number index signature without touching these defaults. Was colony pre-0.5.0, then systems 0.5.x-0.6.x, now metrics aligning with WorldSnapshot.metrics from the universal schema.
